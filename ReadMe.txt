#infos
Sokoban est un jeu vidéo de puzzle inventé au Japon. Sôkoban transcrit avec la méthode Kunrei, désigne un garde d'entrepôt.

Sokoban est un casse tête. Il faut positionner les caisses sur les objectifs verts
J'ai suivi le tuto du site du zéro pour créer ce jeu.
Librairie SDL utilisé pour l'interface.
Ce projet date de 2008

utiliser les flèches pour déplacer mario.
Page pre/suivant -> permet de changer de niveau

avec la sourie il est possible d'ajouter (gauche) ou d'enlever (droit) des marqueurs.
Les marqueurs empêchent les déplacements par erreur.
